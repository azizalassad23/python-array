PYTHON LISTS

1. CREATE 2. ACCESS 3. MODIFY 4. COUNT 5. LOOP >> TERMINAL

01. Membuat List

List adalah variabel super yang bisa menampung banyak data. Gunakan kurung siku [].

items = ["Pedang", "Perisai", "Potion"]

02. Akses Data (Index)

Data pertama ada di index 0. Data terakhir bisa diakses dengan index -1.

print(items[0]) # Output: Pedang

03. Menambah & Menghapus

Gunakan .append() untuk menambah ke belakang, dan .remove() untuk menghapus item.

items.append("Peta") items.remove("Perisai")

04. Menghitung Data (Length)

Seringkali kita perlu tahu "ada berapa item di dalam list?". Gunakan fungsi len().

jumlah = len(items) print(jumlah) # Output akan berupa angka (integer)

05. Looping

Gunakan for loop untuk memproses semua item satu per satu secara otomatis.

for x in items: print("Item: " + x)
